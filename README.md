📚 Smart Study Planner – Productivity App

🌐 Live Website:

🌸 Overview

Smart Study Planner is a pastel-themed productivity web app designed for students to organize their academic life efficiently. The app allows users to manage timetables, assignments, quizzes, and study sessions while automatically generating optimized study schedules based on deadlines and priorities.

Built with modern frontend technologies, the app focuses on clean UI, soft pastel aesthetics, and a user-friendly experience to make studying feel organized and stress-free.

✨ Features

📅 Timetable Creator (Add / Edit / Delete Classes)

📝 Assignment & Project Tracker with Due Dates

🧠 Quiz & Exam Scheduler

⏳ Smart Study Time Generator (Auto Allocation)

📊 Progress Tracking Dashboard

🔔 Reminder Notifications

🕒 Focus Mode with Pomodoro Timer

📈 Subject-wise Analytics

⭐ Priority Tagging (Low / Medium / High)

📆 Monthly & Weekly Calendar View

📌 Daily Task Overview

📓 Quick Notes Section

🔥 Study Streak Tracker

👤 User Profile Setup

🌈 Pastel Aesthetic UI with Soft Card Layout
